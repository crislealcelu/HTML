## Cómo ejecutar la DEMO
1. Abre `index.html` o `login.html` en navegador Chrome.
2. Inicia sesión con las credenciales de arriba.

Todo el estado (saldo, transacciones, contactos) se guarda en `localStorage` del navegador.

## Archivos principales
- `login.html` → Inicio de sesión
- `menu.html` → Dashboard principal
- `deposit.html` → Realizar depósitos
- `sendmoney.html` → Enviar dinero + gestión de contactos con jQuery
- `transactions.html` → Historial completo + filtros + export CSV
